Create a Personal Account

You can create a personal account on our website by following these steps:

  1. Access the site via any internet browser.
  2. From the home page, click "Create" to begin the account creation process.
  3. The "Create a new account" page appears. Enter the required data:
    • Personal information includes first name, last name, and email.
    • Login information: password. Ensure both fields are registered and contain the same content.
  4. Click "Register" to complete the process.

Log in and Create Account

You can also create a personal account by logging in and following these steps:

  1. Log in to the site using any browser and navigate to the home page.
  2. Click "Create" to initiate the account creation process.
  3. The "Create a new account" page will appear. Enter the required data:
    • Personal information includes first name, last name, and email.
    • Login information: password and confirm password. Both fields must be registered and contain the same content.
  4. Click "Register" to complete the process and create your account.
